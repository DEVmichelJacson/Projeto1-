# SKYFIT — Site da Academia

Projeto simples em HTML, CSS e JavaScript para a academia fictícia SKYFIT.

Conteúdo:
- `index.html` — página principal
- `css/styles.css` — estilos
- `js/main.js` — interatividade básica

Como usar localmente:
1. Abra o arquivo `index.html` no navegador (duplo-clique ou arraste para o navegador).

Publicar no GitHub:
1. Inicialize o repositório (se ainda não):

```powershell
git init
git add .
git commit -m "Primeiro commit: site SKYFIT"
```

2. Crie um repositório no GitHub e conecte-o (substitua pela sua URL):

```powershell
git branch -M main
git remote add origin https://github.com/<seu-usuario>/<seu-repositorio>.git
git push -u origin main
```

Para publicar como página estática (GitHub Pages): defina a branch `main` ou `gh-pages` nas configurações do repositório (Settings → Pages) apontando para a pasta raiz.
